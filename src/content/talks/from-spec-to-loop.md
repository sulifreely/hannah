## 为什么 Spec 会失效

强模型时代，重流程的边际收益在下降。

+++

- 过去：用详尽的 Spec 约束弱模型的发挥
- 现在：模型能力溢出，文档成了高成本负担
- 真正稀缺的，是清晰的目标与可验证的结果

+++

2. 反馈怎么接回来

+++

3. 经验怎么留给下一轮 Agent

## Loop 的核心
