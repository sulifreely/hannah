编程 17（进阶）：最长公共前缀 ​

题目 ​

编写一个函数来查找字符串数组中的最长公共前缀。如果不存在公共前缀，返回空字符串 ""。

示例 1：

输入：strs = ["flower","flow","flight"]
输出："fl"
1
2

示例 2：

输入：strs = ["dog","racecar","car"]
输出：""
解释：输入不存在公共前缀。
1
2
3

要求：

  1. 实现纵向扫描法（最直观）
  2. 实现横向扫描法（两两比较）
  3. （拓展）实现分治法和二分查找法
  4. 分析各种方法的时间空间复杂度

参考答案 ​

方法一：纵向扫描法（推荐，最直观） ​

思路：从第一个字符开始，一列一列地检查所有字符串在该位置的字符是否相同，直到出现不同为止。

js
function longestCommonPrefix(strs) {
  if (strs.length === 0) return "";
  
  // 以第一个字符串为基准
  for (let i = 0; i < strs[0].length; i++) {
    const c = strs[0][i];
    // 检查其他所有字符串的第 i 位
    for (let j = 1; j < strs.length; j++) {
      // 如果某个字符串长度不够，或者字符不同，返回前 i 个字符
      if (i === strs[j].length || strs[j][i] !== c) {
        return strs[0].slice(0, i);
      }
    }
  }
  
  return strs[0]; // 第一个字符串就是公共前缀
}

  • 空间 O(1)：只用了常数额外空间。

方法二：横向扫描法（两两比较） ​

思路：先取第一个字符串作为当前公共前缀，然后依次和后面每个字符串比较，找出两者的公共前缀，更新当前公共前缀；如果中途公共前缀变空，直接返回。

js
function longestCommonPrefix(strs) {
  if (strs.length === 0) return "";
  
  let prefix = strs[0];
  
  for (let i = 1; i < strs.length; i++) {
    prefix = commonPrefix(prefix, strs[i]);
    if (prefix === "") break; // 已经没有公共前缀了，提前结束
  }
  
  return prefix;
}

// 求两个字符串的公共前缀
function commonPrefix(s1, s2) {
  let i = 0;
  while (i < s1.length && i < s2.length && s1[i] === s2[i]) {
    i++;
  }
  return s1.slice(0, i);
}

对比纵向扫描：

  • 纵向扫描在「公共前缀很短」时更快（第一个字符就不一样，直接返回）。
  • 横向扫描在「有一个字符串很短」时更快（遇到短字符串公共前缀直接被截断）。
  • 最坏情况一样，都是 O(S)。面试写纵向扫描更直观。

方法三：分治法（拓展） ​

思路：把字符串数组分成左右两部分，分别求左右两部分的最长公共前缀，再求这两个前缀的公共前缀。

js
function longestCommonPrefix(strs) {
  if (strs.length === 0) return "";
  return divideAndConquer(strs, 0, strs.length - 1);
}

function divideAndConquer(strs, left, right) {
  if (left === right) return strs[left]; // 只有一个字符串，它自己就是公共前缀
  
  const mid = (left + right) >> 1;
  const leftPrefix = divideAndConquer(strs, left, mid);
  const rightPrefix = divideAndConquer(strs, mid + 1, right);
  
  return commonPrefix(leftPrefix, rightPrefix);
}

function commonPrefix(s1, s2) {
  let i = 0;
  while (i < s1.length && i < s2.length && s1[i] === s2[i]) {
    i++;
  }
  return s1.slice(0, i);
}

方法四：二分查找法（拓展） ​

思路：最长公共前缀的长度一定不会超过最短字符串的长度。对「长度」进行二分：如果 mid 长度的前缀是公共的，就尝试更长；否则尝试更短。

js
function longestCommonPrefix(strs) {
  if (strs.length === 0) return "";
  
  // 找到最短字符串的长度（公共前缀最长不会超过它）
  let minLen = Infinity;
  for (const s of strs) {
    minLen = Math.min(minLen, s.length);
  }
  
  let low = 0;
  let high = minLen;
  
  while (low < high) {
    // 注意：向上取整，避免死循环（因为 low = mid 时要向上取整）
    const mid = low + ((high - low + 1) >> 1);
    if (isCommonPrefix(strs, mid)) {
      low = mid; // mid 长度的前缀是公共的，尝试更长
    } else {
      high = mid - 1; // 不是公共的，要更短
    }
  }
  
  return strs[0].slice(0, low);
}

// 检查所有字符串的前 len 个字符是否都相同
function isCommonPrefix(strs, len) {
  const prefix = strs[0].slice(0, len);
  for (let i = 1; i < strs.length; i++) {
    if (!strs[i].startsWith(prefix)) {
      return false;
    }
  }
  return true;
}

为什么是向上取整 low + ((high - low + 1) >> 1)？ 因为当 low = mid 的写法时，如果 mid 向下取整，当 low = high - 1 时 mid = low，会陷入死循环。向上取整确保每次循环区间都在缩小。

复杂度：

  • 时间 O(S × log m)：m 是最短字符串的长度，二分 log m 次，每次检查 O(S) 个字符。
  • 空间 O(1)。

追问 1：如果要求「最长公共子序列」（LCS）而不是「公共前缀」，思路有什么不同？ ​

答 ​

公共前缀是「从第一个字符开始连续相同」，公共子序列是「字符按顺序出现即可，不一定连续」。两者完全不是一个难度等级。

最长公共子序列（LCS）经典 DP 解法 ​

dp[i][j] 表示 s1 前 i 个字符和 s2 前 j 个字符的 LCS 长度

如果 s1[i-1] === s2[j-1]：
  dp[i][j] = dp[i-1][j-1] + 1
否则：
  dp[i][j] = max(dp[i-1][j], dp[i][j-1])

js
function longestCommonSubsequence(text1, text2) {
  const m = text1.length;
  const n = text2.length;
  const dp = Array.from({ length: m + 1 }, () => new Array(n + 1).fill(0));
  
  for (let i = 1; i <= m; i++) {
    for (let j = 1; j <= n; j++) {
      if (text1[i - 1] === text2[j - 1]) {
        dp[i][j] = dp[i - 1][j - 1] + 1;
      } else {
        dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]);
      }
    }
  }
  
  return dp[m][n];
}

追问 2：如果字符串数量非常大（比如 100 万条），如何高效找最长公共前缀？可以用什么数据结构？ ​

答 ​

用 Trie（字典树 / 前缀树）：

思路：

  1. 把所有字符串插入到 Trie 中。
  2. 从根节点开始往下走，只要当前节点有且只有一个子节点（且不是某个字符串的结尾），就说明所有字符串在这一位都相同，继续走；
  3. 当节点有多个子节点，或者到达某个字符串结尾时，停止。走过的路径就是最长公共前缀。
js
class TrieNode {
  constructor() {
    this.children = new Map(); // 字符 -> 子节点
    this.isEnd = false;        // 是否是某个单词的结尾
  }
}

function longestCommonPrefix(strs) {
  if (strs.length === 0) return "";
  
  // 1. 构建 Trie
  const root = new TrieNode();
  for (const s of strs) {
    if (s === "") return ""; // 有空字符串，公共前缀一定是空
    let node = root;
    for (const c of s) {
      if (!node.children.has(c)) {
        node.children.set(c, new TrieNode());
      }
      node = node.children.get(c);
    }
    node.isEnd = true;
  }
  
  // 2. 从根往下找最长公共前缀
  let prefix = "";
  let node = root;
  while (node.children.size === 1 && !node.isEnd) {
    const [char, child] = node.children.entries().next().value;
    prefix += char;
    node = child;
  }
  
  return prefix;
}

  • 构建 Trie 时间 O(S)（S 是所有字符总数）。
  • 查找公共前缀时间 O(L)（L 是公共前缀长度）。

拓展：如果是分布式系统中找最长公共前缀，可以用 MapReduce：先按前缀哈希分片，每台机器找自己分片内的公共前缀，再汇总。

追问 3：「最长公共前缀」在实际工程中有哪些应用场景？ ​

答 ​

  1. 搜索框自动补全（Autocomplete） 你输入 "app"，搜索框提示 "apple"、"application"。背后就是 Trie 前缀树，找公共前缀并推荐最热门的补全项。

  2. IP 路由最长前缀匹配（Longest Prefix Match） 路由器转发表中，一个 IP 地址可能匹配多条路由规则，选择前缀最长的那条。这是计算机网络的经典问题，常用 Trie / 基数树实现。

  3. 字符串排序 / 字典序分组 按前缀对单词进行分组（比如通讯录按姓氏拼音首字母分组）。

  4. 文件系统路径 找多个文件路径的公共前缀（共同的父目录）。比如 git diff 中找变更文件的公共祖先目录。

  5. 生物信息学 DNA 序列比对中找公共前缀、基因序列的相似区域识别。

  6. 压缩算法 LZ77、LZW 等字典压缩算法利用「已出现过的前缀」来压缩数据。

  7. 拼写检查 找与输入单词前缀相同的候选词，作为拼写错误的建议。

简单总结：凡是涉及「按前缀查找 / 分组 / 匹配」的场景，最长公共前缀和 Trie 树都有用武之地。
